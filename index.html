WindowJS

Vanilla JavaScript plugin for creating complex windows with a minimum of code.

Getting started

To start using this plugin, you just need to download it and include three files.

HTML

<!DOCTYPE html>
<html>
	<head>
		...
		<link rel="stylesheet" type="text/css" href="css/window_light.css" /> <!-- OR "css/window_dark.css" -->
		<link rel="stylesheet" type="text/css" href="css/fontello.css" />
		...
	</head>

	<body>
		...
		<script type="text/javascript" src="js/window.js"></script>
	</body>
</html>

JavaScript

var w1 = w2 = w3 = w4 = w5 = null;

document.getElementById('open1').addEventListener('click', function(){
	var options = {
		title: 'Window with Ajax content inside !',
		ajaxContent: {
			url: './simple.html',
			method: 'get'
		},
		onClose: function(){
			w1 = null;
		}
	};
	
	w1 = w1 instanceof WindowJS ? w1 : new WindowJS(options);
});


document.getElementById('open2').addEventListener('click', function(){
	var options = {
		width: 700,
		height: 270,
		title: 'Not resizable window !',
		resizable: false,
		content: '<h1>Not resizable window with custom size (width/height)</h1><p>Just another paragraph.</p>',
		onClose: function(){
			w2 = null;
		}
	};
	
	w2 = w2 instanceof WindowJS ? w2 : new WindowJS(options);
});


document.getElementById('open3').addEventListener('click', function(){
	var options = {
		title: 'Default window !',
		content: '<h1>Default Window</h1><p>With two custom options ("title" and "content")</p>',
		onClose: function(){
			w3 = null;
		}
	};
	
	w3 = w3 instanceof WindowJS ? w3 : new WindowJS(options);
});

document.getElementById('open4').addEventListener('click', function(){
	var options = {
		title: 'Modal window !',
		modal: true,
		content: '<h1>Modal window</h1><p>The content of the 4th window</p><p>And another paragraph :)</p>',
		onClose: function(){
			w4 = null;
		}
	};
	
	w4 = w4 instanceof WindowJS ? w4 : new WindowJS(options);
});

document.getElementById('open5').addEventListener('click', function(){
	var options = {
		width: 940,
		height: 715,
		title: 'Wikipedia !',
		iframe: 'https://fr.wikipedia.org/',
		maximized: false,
		contentPadding: false,
		onClose: function(){
			w5 = null;
		}
	};
	
	w5 = w5 instanceof WindowJS ? w5 : new WindowJS(options);
});

document.getElementById('open6').addEventListener('click', function(){
	new WindowJS();
});

Options

Name Type Default value Accepted value Desctiption
width integer 700 any integer The width of the window (in pixels)
height integer 500 any integer The height of the window (in pixels)
minWidth integer 225 any integer The minimum width of the window reduction (in pixels)
minHeight integer 100 any integer The minimum height of the window reduction (in pixels)
title string 'Title Here !' any string The title of the window
modal boolean false true or false Defines the type of window. If set to true, then the window will be modal if set to false the window will be normal.
modal boolean false true or false Defines the type of window. If set to true, then the window will be modal if set to false the window will be normal.
resizable boolean true true or false Allows you to make the window resizable.
draggable boolean true true or false Allows you to make the window draggable.
onDragOpacity integer 85 any integer Represent the percentage of opacity when the window is moved.
reducedWidth integer 225 any integer The width of the reduced window (in pixels)
maximized boolean false true or false If set to true, display the window on the entire page, if set to false the display window with "width / height" values
overlayColor string null hexadecimal color Set the color of the overlay when the window is modal. If not specified, the value is taken from the css file
overlayOpacity integer null any integer Represent the percentage of opacity for the overlay. If not specified, the value is taken from the css file.
contentPadding boolean true true or false If set to false the content wrapper use the padding of ".w-content" class contains in "windows.css" file. If set to true the content wrapper does not have paddings.
loaderImg string './images/w-loader.gif' path string The relative path to the loader image.
ajaxContent object null {
    url: 'the url string',
    method: 'post | get'
    data: 'key=value'
} 		Load the window content from an ajax query.
url: 'someurl.com'
method: 'post' or 'get'
data: 'key=value&key2=value2...'
iframe string null src url string If a source url is specified, the window content will be an <iframe> with that source.
content string 'Content of the window Here !' text or html string The content of the window.

Callback functions

Name Type Default value Accepted value Desctiption
onOpen function null any valid function() Runs when the window is opened.
onClose function null any valid function() Runs when the window is closed.
onMinimize function null any valid function() Runs when the window is minimized.
onMaximize function null any valid function() Runs when the window is maximized.
onEnlarge function null any valid function() Runs when the window is enlarged.
onReduce function null any valid function() Runs when the window is reduced.
onShow function null any valid function() Runs when the window body (content) is shown.
onHide function null any valid function() Runs when the window body (content) is hidden.
onDragStart function null any valid function() Runs at the start of window dragging.
onDragEnd function null any valid function() Runs at the end of window dragging.
onDragging function null any valid function() Runs during the window dragging.
onResizeStart function null any valid function() Runs at the start of window resizing.
onResizeEnd function null any valid function() Runs at the end of window resizing.
onResizing function null any valid function() Runs during the window resizing.

Themes

There are only two themes in the current version (light and dark). Use one of the below css files in your projects

HTML

<link rel="stylesheet" type="text/css" href="css/window_light.css" />
<!-- OR -->
<link rel="stylesheet" type="text/css" href="css/window_dark.css" />

Styles

Styles can be changed in css files. (window_light.css or window_dark.css - depend on which one you use)
